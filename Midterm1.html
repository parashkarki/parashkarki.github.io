PART I

Foundational Concepts and Definitions

Understanding linear systems, the inverse problem, consistent/inconsistent systems, and perspectives on solving

Section 1.1: True/False Questions

1
In a linear system, an allowed operation includes multiplying unknowns by each other (e.g., \(xy = 4\)).
True/False
Answer: FALSE ✗
Why This Is False
Multiplying unknowns by each other creates a nonlinear term. Linear systems require:
  • Unknowns multiplied ONLY by constants (e.g., \(5x, 3y, -2z\))
  • Sums of these terms only
The expression \(xy = 4\) is a second-degree term (nonlinear) and violates the definition of linearity.
Fundamental Rule: "Linear" means first-degree terms only. NO products of unknowns allowed.
2
The definition of a "Linear" system requires that unknowns are multiplied only by constant numbers and involves sums only.
True/False
Answer: TRUE ✓
Why This Is True
This is the precise, formal mathematical definition of linearity. A linear equation always has the form:
\(a_1x_1 + a_2x_2 + \cdots + a_nx_n = b\)
where:
  • \(a_1, a_2, \ldots, a_n, b\) are constants
  • \(x_1, x_2, \ldots, x_n\) are unknowns
  • Only additions and multiplications by constants are used
This definition captures the essence of linearity: constant coefficients times unknowns, added together.
3
The "Inverse Problem" is defined as the challenge of discovering the unknown input when the output and the system are already known.
True/False
Answer: TRUE ✓
Understanding the Inverse Problem
The inverse problem is fundamental to linear algebra:
Given: Matrix \(A\) (the system) and vector \(\mathbf{b}\) (the output)
Find: Vector \(\mathbf{x}\) (the input) such that \(A\mathbf{x} = \mathbf{b}\)
Contrast With Forward Problem
Forward: Given \(A\) and \(\mathbf{x}\), compute \(A\mathbf{x} = \mathbf{b}\)
Inverse: Given \(A\) and \(\mathbf{b}\), solve for \(\mathbf{x}\)
Central Problem: The inverse problem is the PRIMARY focus of linear algebra courses because it solves real-world problems.
4
A "Consistent System" is defined as a system that has at least one solution, meaning the target is reachable.
True/False
Answer: TRUE ✓
Definition of Consistent
A linear system \(A\mathbf{x} = \mathbf{b}\) is consistent if at least one solution exists. This means the target vector \(\mathbf{b}\) is reachable as a linear combination of the columns of \(A\).
In Column Space Terms
A system is consistent if and only if \(\mathbf{b} \in C(A)\) (the column space of \(A\)).
Possible Cases
Consistent: Exactly one solution OR infinitely many solutions
Inconsistent: No solutions at all
5
In modern linear algebra, the "Old Question" was computing the output given an input \(\mathbf{x}\), while the "New Question" focuses on solving for \(\mathbf{x}\).
True/False
Answer: TRUE ✓
The Two Questions
Old Question (Forward Problem): Given \(A\) and \(\mathbf{x}\), compute \(A\mathbf{x} = \mathbf{b}\)
New Question (Inverse Problem): Given \(A\) and \(\mathbf{b}\), find \(\mathbf{x}\) such that \(A\mathbf{x} = \mathbf{b}\)
Why This Shift Matters
Real applications require solving the inverse problem:
  • Given a desired output, find the input needed
  • This is more complex and practically important
  • This is why we study solving systems, not just multiplying matrices
6
Forbidden operations in linear algebra include powers (e.g., \(x^2\)) and trigonometric functions (e.g., \(\cos(y)\)).
True/False
Answer: TRUE ✓
Why These Operations Are Forbidden
Any nonlinear operation destroys the fundamental structure that makes linear algebra tractable. Forbidden operations include:
Examples of Nonlinear Operations
  • Powers: \(x^2, y^3, z^{1/2}, \sqrt{x}\)
  • Trigonometric: \(\sin(x), \cos(y), \tan(z)\)
  • Exponential: \(e^x, 2^y, 10^z\)
  • Logarithmic: \(\log(x), \ln(y)\)
  • Products of unknowns: \(xy, xz, yz\)
Linearity is restrictive, but this restriction is what allows us to solve systems algebraically and use matrix methods.
7
An "Inconsistent System" occurs when the equations contradict each other, resulting in no solution.
True/False
Answer: TRUE ✓
Definition of Inconsistency
An inconsistent system has NO solution because the equations contradict each other. No values of the unknowns can satisfy all equations simultaneously.
Classic Example
\(x + y = 1\)
\(x + y = 2\)
These two equations contradict each other. No \(x, y\) exist that satisfy both. The lines are parallel.
In Column Space Terms
The system is inconsistent when \(\mathbf{b} \notin C(A)\) (the target is outside the column space).
8
The "Solution Set" is the collection of all possible values for the unknowns that make every equation in the system true simultaneously.
True/False
Answer: TRUE ✓
Definition of Solution Set
The solution set contains ALL vectors \(\mathbf{x}\) that satisfy the system \(A\mathbf{x} = \mathbf{b}\). Every component of \(\mathbf{x}\) must satisfy EVERY equation simultaneously.
Possible Forms of Solution Sets
  • Empty set: No solutions (inconsistent system)
  • Single point: Exactly one solution (unique solution)
  • Line: Infinitely many solutions along a 1D line (free variables)
  • Plane: Infinitely many solutions in a 2D plane (more free variables)
  • Higher-dimensional: Infinitely many solutions in higher dimensions
The solution set is our answer to the inverse problem. It's the complete answer to the question "What inputs reach the target?"
9
In the "Column Picture," we view the system as a combination of columns weighted by the components of \(\mathbf{x}\).
True/False
Answer: TRUE ✓
The Column Picture Perspective
In the column picture, we rewrite \(A\mathbf{x} = \mathbf{b}\) as:
\(x_1 \text{col}_1(A) + x_2 \text{col}_2(A) + \cdots + x_n \text{col}_n(A) = \mathbf{b}\)
What This Means
Each component \(x_i\) of the input vector \(\mathbf{x}\) acts as a WEIGHT or COEFFICIENT for the corresponding column of \(A\). We scale each column and add them up to reach the target \(\mathbf{b}\).
The Column Picture Question
"What combination (linear combination) of the columns of \(A\) produces the vector \(\mathbf{b}\)?" The answer is the input vector \(\mathbf{x}\).
Central Insight: The column picture is a visual way to think about linear combinations. Solving the system means finding the right weights for the columns.
10
A system is a collection of linear equations involving the same set of variables.
True/False
Answer: TRUE ✓
Definition of a System
A system must contain multiple linear equations where ALL equations involve the SAME unknowns. This is what allows us to solve for the unknowns.
Example of a System
\(2x + 3y = 5\)
\(x - y = 1\)
\(3x + 2y = 4\)
This is a system of 3 equations in 2 unknowns. Every equation involves \(x\) and \(y\).
Matrix Representation
Any system can be written as \(A\mathbf{x} = \mathbf{b}\) where:
  • \(A\) is the coefficient matrix (containing the system structure)
  • \(\mathbf{x}\) is the unknown vector
  • \(\mathbf{b}\) is the target vector

Section 1.2: Multiple-Choice Questions: Anatomy of the Machine

1
In the matrix equation \(Ax = b\), which component represents the "Structure" or the "Machine" itself?
Multiple Choice
A) The input vector
B) The output vector
C) The matrix ✓
D) The scalar weights
Solution
The Machine Metaphor
Think of the system \(A\mathbf{x} = \mathbf{b}\) as a machine:
  • Input (\(\mathbf{x}\)): What goes into the machine (variable)
  • Machine (\(A\)): The structure and mechanism (fixed)
  • Output (\(\mathbf{b}\)): What comes out (the result)
Why A Is the Machine
The matrix \(A\) is fixed and completely defines the system. It contains all the coefficients that determine how the input is transformed into the output. Different machines (different matrices) transform inputs differently.
The correct answer is C: The matrix A is the machine/structure. It's what transforms \(\mathbf{x}\) into \(\mathbf{b}\).
2
For an \(m \times n\) matrix, the variable \(m\) specifically represents the number of:
Multiple Choice
A) Columns
B) Rows ✓
C) Unknowns
D) Dimensions in the input space
Solution
Matrix Dimension Convention
An \(m \times n\) matrix always means:
  • \(m\) = number of ROWS (the first dimension)
  • \(n\) = number of COLUMNS (the second dimension)
This is the standard convention: rows first, then columns.
In System Context
For a system \(A\mathbf{x} = \mathbf{b}\) with an \(m \times n\) matrix \(A\):
  • \(m\) = number of EQUATIONS (one per row)
  • \(n\) = number of UNKNOWNS (one per column)
  • \(\mathbf{b} \in \mathbb{R}^m\) (m components, one from each equation)
  • \(\mathbf{x} \in \mathbb{R}^n\) (n components, one for each unknown)
The correct answer is B: m represents the number of rows.
3
For an \(m \times n\) matrix, the variable \(n\) specifically represents the number of:
Multiple Choice
A) Equations
B) Output components
C) Columns ✓
D) Target vectors
Solution
Understanding n
\(n\) is the second dimension and represents the number of COLUMNS.
Columns Represent Unknowns
Each column of \(A\) corresponds to ONE unknown variable:
  • Column 1 has coefficients for \(x_1\)
  • Column 2 has coefficients for \(x_2\)
  • And so on...
Therefore: \(\mathbf{x} \in \mathbb{R}^n\) (n unknowns)
The correct answer is C: n represents the number of columns, which equals the number of unknowns.
4
In the Column Picture perspective, the components of the input vector (e.g., \(x_1, x_2, \ldots, x_n\)) are mathematically treated as:
Multiple Choice
A) The targets
B) The weights (coefficients) for the columns ✓
C) The dot product results
D) The pivots
Solution
The Column Picture Formula
The column picture rewrites \(A\mathbf{x} = \mathbf{b}\) as:
\(x_1 \text{col}_1(A) + x_2 \text{col}_2(A) + \cdots + x_n \text{col}_n(A) = \mathbf{b}\)
x Components as Weights
In this formula, each \(x_i\) is a SCALAR WEIGHT that multiplies (scales) the corresponding column. The components of \(\mathbf{x}\) are the weights we need to combine the columns into \(\mathbf{b}\).
The correct answer is B: Components of x are weights/coefficients for the columns.
5
If a matrix has 3 rows and 2 columns, how is it classified in terms of dimensions?
Multiple Choice
A) \(2 \times 3\)
B) \(3 \times 2\) ✓
C) \(3 \times 3\)
D) \(2 \times 2\)
Solution
The Convention
Matrix dimensions are ALWAYS written as:
(number of rows) × (number of columns)
Applying the Convention
Given: 3 rows and 2 columns
Therefore: \(3 \times 2\) matrix

NOT \(2 \times 3\) (that would be 2 rows and 3 columns)
The correct answer is B: \(3 \times 2\). Remember: rows × columns.
6
In the fundamental equation \(Ax = b\), the vector \(\mathbf{b}\) is referred to as the:
Multiple Choice
A) Unknown input
B) System structure
C) Known output (Target) ✓
D) Transformation weight
Solution
The Role of b
\(\mathbf{b}\) is the OUTPUT or TARGET of the system. It's the result we get when we apply the transformation \(A\) to the input \(\mathbf{x}\).
In Different Contexts
In the forward problem: We know \(\mathbf{x}\) and \(A\), and we compute \(\mathbf{b}\).
In the inverse problem: We know \(\mathbf{b}\) and \(A\), and we find \(\mathbf{x}\).
Why "Target"?
We think of \(\mathbf{b}\) as a target we want to reach. The question "Is the system solvable?" means "Can we reach the target \(\mathbf{b}\)?" by choosing the right input \(\mathbf{x}\).
The correct answer is C: b is the known output (target).
7
In the perspective of "Perspective 1: The Row Picture," what mathematical operation is primarily used to find each component of \(\mathbf{b}\)?
Multiple Choice
A) Vector addition
B) Linear combination
C) Dot product ✓
D) Matrix inversion
Solution
The Row Picture Operation
In the row picture, we process the system one row at a time. Each row of \(A\) is a vector that we dot with the input vector \(\mathbf{x}\):
\(b_i = \text{row}_i(A) \cdot \mathbf{x}\)
Why Dot Product?
Each equation in the original system is a dot product:
\(a_{i1}x_1 + a_{i2}x_2 + \cdots + a_{in}x_n = b_i\)
This is precisely the dot product of \([\text{row}_i]\) with \(\mathbf{x}\).
The correct answer is C: Dot product. Each row dotted with x gives one output component.
8
Which perspective of \(Ax = b\) is described as "looking at one equation at a time"?
Multiple Choice
A) The Column Picture
B) The Null Space
C) The Row Picture ✓
D) The Echelon Form
Solution
The Row Picture Definition
The row picture examines the system one equation (one row) at a time. Each row represents one linear equation or constraint.
Visualization
For 2 unknowns: Each equation is a line. We look for where all lines intersect.
For 3 unknowns: Each equation is a plane. We look for where all planes intersect.
For n unknowns: Each equation is a hyperplane. The intersection is the solution set.
Key Distinction: Row picture: equation by equation. Column picture: all columns together.
The correct answer is C: The Row Picture is about looking at equations one at a time.
9
Which perspective views the system as "Which combination of columns produces the particular vector \(\mathbf{b}\)?"
Multiple Choice
A) The Row Picture
B) The Column Picture ✓
C) The Inverse Problem
D) The Determinant
Solution
The Column Picture Question
The column picture asks exactly: "What combination (linear combination) of the columns of \(A\) produces the vector \(\mathbf{b}\)?"
The Column Picture Formula
We rewrite the system as:
\(x_1 \text{col}_1(A) + x_2 \text{col}_2(A) + \cdots + x_n \text{col}_n(A) = \mathbf{b}\)
The answer \(\mathbf{x}\) tells us the weights needed.
The correct answer is B: The Column Picture views the system as finding a linear combination of columns.
10
In the "Anatomy of the Machine" (Slide 5-1), Column 1 of matrix \(A\) contains:
Multiple Choice
A) The constants from the right-hand side of the equations
B) The coefficients of the first unknown (\(x_1\)) across all equations ✓
C) The solutions to the first equation
D) The identity elements
Solution
How Matrices Are Structured
Matrix \(A\) is organized by UNKNOWNS (columns), not by equations (rows):
  • Column 1 = all coefficients of \(x_1\)
  • Column 2 = all coefficients of \(x_2\)
  • Column n = all coefficients of \(x_n\)
Example
If the system is:
\(2x_1 + 3x_2 = 5\)
\(x_1 - x_2 = 1\)
Then Column 1 of \(A\) is \([2, 1]^T\) (the coefficients of \(x_1\))
The correct answer is B: Column 1 contains the coefficients of x₁ across all equations.

Section 1.3: Calculation-Based Questions: The Row Picture

1
Given Row 1 = \([2 \; 5]\) and input vector \(\mathbf{x} = \begin{bmatrix} x_1 \\ x_2 \end{bmatrix}\), write the algebraic expression for the first component of the output, \(b_1\).
Calculation
Solution
Step 1: Identify the Operation
The first component \(b_1\) is the result of dotting Row 1 with input \(\mathbf{x}\):
\(b_1 = [2 \; 5] \cdot \begin{bmatrix} x_1 \\ x_2 \end{bmatrix}\)
Step 2: Compute the Dot Product
\(b_1 = 2(x_1) + 5(x_2)\)
Final Answer
\(b_1 = 2x_1 + 5x_2\)
This is the algebraic form of the first equation. If this came from a full system, we'd set \(b_1\) equal to some constant (e.g., \(2x_1 + 5x_2 = 7\)).
2
Given Row 2 = \([3 \; 7]\) and input vector \(\mathbf{x} = \begin{bmatrix} 1 \\ 2 \end{bmatrix}\), compute the numerical value of \(b_2\).
Calculation
Solution
Step 1: Set Up the Dot Product
\(b_2 = [3 \; 7] \cdot \begin{bmatrix} 1 \\ 2 \end{bmatrix}\)
Step 2: Multiply Each Component
First component: \(3 \times 1 = 3\)
Second component: \(7 \times 2 = 14\)
Step 3: Add the Results
\(b_2 = 3 + 14 = 17\)
Final Answer
\(b_2 = 17\)
3
Given Row 1 = \([2 \; 1]\) and \(\mathbf{x} = \begin{bmatrix} 4 \\ 5 \end{bmatrix}\), compute \(b_1\).
Calculation
Solution
Step 1: Dot Product
\(b_1 = [2 \; 1] \cdot \begin{bmatrix} 4 \\ 5 \end{bmatrix}\)
Step 2: Compute
\(b_1 = 2(4) + 1(5) = 8 + 5 = 13\)
Final Answer
\(b_1 = 13\)
4
Given Row 2 = \([3 \; 0]\) and \(\mathbf{x} = \begin{bmatrix} 4 \\ 5 \end{bmatrix}\), compute \(b_2\).
Calculation
Solution
Step 1: Dot Product
\(b_2 = [3 \; 0] \cdot \begin{bmatrix} 4 \\ 5 \end{bmatrix}\)
Step 2: Compute
\(b_2 = 3(4) + 0(5) = 12 + 0 = 12\)
Final Answer
\(b_2 = 12\)
Note: The zero coefficient means the second component of x doesn't affect the result. Only the first component (4) contributes to b₂.
5
Given Row 3 = \([1 \; 2]\) and \(\mathbf{x} = \begin{bmatrix} 4 \\ 5 \end{bmatrix}\), compute \(b_3\).
Calculation
Solution
Step 1: Dot Product
\(b_3 = [1 \; 2] \cdot \begin{bmatrix} 4 \\ 5 \end{bmatrix}\)
Step 2: Compute
\(b_3 = 1(4) + 2(5) = 4 + 10 = 14\)
Final Answer
\(b_3 = 14\)
6
Consider the matrix \(A = \begin{bmatrix} 1 & 2 & 1 \\ 1 & 1 & 5 \\ 2 & 1 & 2 \end{bmatrix}\). If \(\mathbf{x} = \begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix}\), calculate \(A\mathbf{x} = \mathbf{b_1}\).
Calculation
Solution
Step 1: Compute First Component
\(b_1 = [1 \; 2 \; 1] \cdot \begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix} = 1(1) + 2(0) + 1(1) = 1 + 0 + 1 = 2\)
Step 2: Compute Second Component
\(b_2 = [1 \; 1 \; 5] \cdot \begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix} = 1(1) + 1(0) + 5(1) = 1 + 0 + 5 = 6\)
Step 3: Compute Third Component
\(b_3 = [2 \; 1 \; 2] \cdot \begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix} = 2(1) + 1(0) + 2(1) = 2 + 0 + 2 = 4\)
Final Answer
\(A\mathbf{x} = \mathbf{b_1} = \begin{bmatrix} 2 \\ 6 \\ 4 \end{bmatrix}\)
This shows the complete matrix-vector multiplication using the row picture (dot products).
7
Using matrix from question 6 and \(\mathbf{x} = \begin{bmatrix} 1 \\ 1 \\ 1 \end{bmatrix}\), calculate \(A\mathbf{x} = \mathbf{b_2}\).
Calculation
Solution
Step 1: First Component
\(b_1 = [1 \; 2 \; 1] \cdot \begin{bmatrix} 1 \\ 1 \\ 1 \end{bmatrix} = 1(1) + 2(1) + 1(1) = 1 + 2 + 1 = 4\)
Step 2: Second Component
\(b_2 = [1 \; 1 \; 5] \cdot \begin{bmatrix} 1 \\ 1 \\ 1 \end{bmatrix} = 1(1) + 1(1) + 5(1) = 1 + 1 + 5 = 7\)
Step 3: Third Component
\(b_3 = [2 \; 1 \; 2] \cdot \begin{bmatrix} 1 \\ 1 \\ 1 \end{bmatrix} = 2(1) + 1(1) + 2(1) = 2 + 1 + 2 = 5\)
Final Answer
\(A\mathbf{x} = \mathbf{b_2} = \begin{bmatrix} 4 \\ 7 \\ 5 \end{bmatrix}\)
8
Using matrix from question 6 and \(\mathbf{x} = \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix}\), calculate \(A\mathbf{x} = \mathbf{b_3}\).
Calculation
Solution
Step 1: First Component
\(b_1 = [1 \; 2 \; 1] \cdot \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} = 1(0) + 2(1) + 1(0) = 0 + 2 + 0 = 2\)
Step 2: Second Component
\(b_2 = [1 \; 1 \; 5] \cdot \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} = 1(0) + 1(1) + 5(0) = 0 + 1 + 0 = 1\)
Step 3: Third Component
\(b_3 = [2 \; 1 \; 2] \cdot \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} = 2(0) + 1(1) + 2(0) = 0 + 1 + 0 = 1\)
Final Answer
\(A\mathbf{x} = \mathbf{b_3} = \begin{bmatrix} 2 \\ 1 \\ 1 \end{bmatrix}\)
This result is Column 2 of matrix A! When multiplying by a unit vector with a single 1 in position i, we extract column i.
9
For the row vector \([2 \; 2 \; 0]\) and input \(\mathbf{x} = \begin{bmatrix} 3 \\ 1 \\ 2 \end{bmatrix}\), calculate the resulting dot product.
Calculation
Solution
Step 1: Set Up the Dot Product
\([2 \; 2 \; 0] \cdot \begin{bmatrix} 3 \\ 1 \\ 2 \end{bmatrix}\)
Step 2: Multiply Corresponding Components
First: \(2 \times 3 = 6\)
Second: \(2 \times 1 = 2\)
Third: \(0 \times 2 = 0\)
Step 3: Add the Results
\(6 + 2 + 0 = 8\)
Final Answer
\([2 \; 2 \; 0] \cdot \begin{bmatrix} 3 \\ 1 \\ 2 \end{bmatrix} = 8\)
10
For the row vector \([0 \; 4 \; -2]\) and input \(\mathbf{x} = \begin{bmatrix} 3 \\ 1 \\ 2 \end{bmatrix}\), calculate the resulting dot product.
Calculation
Solution
Step 1: Set Up the Dot Product
\([0 \; 4 \; -2] \cdot \begin{bmatrix} 3 \\ 1 \\ 2 \end{bmatrix}\)
Step 2: Multiply Corresponding Components
First: \(0 \times 3 = 0\)
Second: \(4 \times 1 = 4\)
Third: \((-2) \times 2 = -4\)
Step 3: Add the Results
\(0 + 4 + (-4) = 0 + 4 - 4 = 0\)
Final Answer
\([0 \; 4 \; -2] \cdot \begin{bmatrix} 3 \\ 1 \\ 2 \end{bmatrix} = 0\)
The dot product is 0, which means these two vectors are ORTHOGONAL (perpendicular). This is an important concept: when a dot product equals zero, the vectors are perpendicular.
