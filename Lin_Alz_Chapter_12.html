📌 What is Dimension?

Imagine you're describing how many different independent directions you can move:

  • On a line: Only 1 direction (left/right) → Dimension = 1
  • On a piece of paper: 2 directions (left/right and up/down) → Dimension = 2
  • In the real world: 3 directions (length, width, height) → Dimension = 3
  • In linear algebra: The number of independent basis vectors in a vector space
💡
Key Definition: Dimension is the minimum number of independent vectors needed to represent every point in a space. It's like asking: "How many independent directions describe this space?"

🔗 The Rank & Dimension Connection

Rank = Number of pivot columns in a matrix

Dimension = Number of vectors in a basis of a space

Key Link: Rank = Dimension of Column Space = Dimension of Row Space

Think of it this way: When you reduce a matrix, the pivots show you which columns are independent. These independent columns form a basis, and the count of pivots equals the dimension!

🏗️ The Four Fundamental Subspaces

1️⃣ Column Space C(A)

What it is: All possible outputs when multiplying matrix A by any input vector x

Picture: If A is 3×2, column space is like a plane floating in 3D space

Dimension: = Rank of A (number of pivot columns)

How to find: Use the pivot columns as the basis

2️⃣ Row Space C(A^T)

What it is: All possible linear combinations of the rows of A

Picture: The "output space" created by row combinations

Dimension: = Rank of A (surprisingly, same as column space!)

Why this works: Row rank = Column rank (fundamental theorem!)

3️⃣ Null Space N(A)

What it is: All vectors x where Ax = 0 (inputs that produce zero output)

Picture: "Which inputs completely disappear when transformed?"

Dimension: = n - Rank (where n = total columns)

Formula: Nullity = Free variables = n - Rank

4️⃣ Left Null Space N(A^T)

What it is: All vectors y where y^T A = 0 (vectors that disappear from the left)

Picture: "Which row vectors vanish when hitting A from the left?"

Dimension: = m - Rank (where m = number of rows)

Note: This is the null space of A^T (the transpose)

🔄 The Fundamental Theorem

The Balance Law:

Rank(A) + Nullity(A) = n (number of pivots) + (number of free variables) = (total columns)

This beautiful equation tells us that pivot columns and free columns must add up perfectly!

Another profound truth: The row space and null space are perpendicular (orthogonal) to each other. If a vector is in the null space, it's perpendicular to every row of the matrix.

📊 Matrix Size
Rank ≤ min(m, n)

🧠 Understanding Dimension

Think about everyday examples:

  • A street address: Just need the house number → 1 dimension
  • A city grid: Need avenue and street → 2 dimensions
  • Global position: Need latitude, longitude, altitude → 3 dimensions
  • Data in a spreadsheet: Each independent column is a dimension

In linear algebra: Dimension = number of linearly independent directions

🎨 What is a Basis?

A basis is: A set of vectors that can build any vector in a space

  • Independent: No vector is a combination of others (they don't overlap)
  • Spanning: Together they cover the entire space
  • Minimal: You can't remove any vector without losing coverage
  • Count: A basis always has exactly dimension vectors
💡
Think of it like LEGO: A basis is the minimum set of unique LEGO pieces you need to build anything in that space. You can't use duplicate pieces, and every piece is essential.

🎯 Pivot Columns = Basis

The Magic Fact: The pivot columns of A form a basis for its column space!

  • Why? Pivot columns are independent (they create the identity pattern)
  • Count: Exactly equal to the rank
  • Free columns: These are combinations of pivot columns, not independent
  • Dimension: = Number of pivots

⚡ Orthogonal Subspaces

The Perpendicularity Fact:

  • Row Space ⊥ Null Space (perpendicular!)
  • Column Space ⊥ Left Null Space (perpendicular!)

This means: If you pick any vector from the row space and any from the null space, their dot product is zero!

🎯
Why this matters: These perpendicular relationships are the foundation of solving equations, least squares, and projections. It's pure geometry!

📝 Example 1: Simple 1×3 Matrix

Matrix: A = [1 2 3] (1 row, 3 columns)

Analysis:

1. Row Space C(A^T):

Only one non-zero row: [1, 2, 3]. This creates a line in 3D space.

Dimension = 1 (one pivot)

2. Column Space C(A):

All scalar multiples of the single column [1]. This is the real line.

Dimension = 1 (one pivot)

3. Null Space N(A):

Solve: x₁ + 2x₂ + 3x₃ = 0

We have 1 pivot and 2 free variables, creating a 2D plane in 3D space.

Dimension = 3 - 1 = 2

4. Left Null Space N(A^T):

Solve: y[1 2 3] = 0 → y = 0

Dimension = 1 - 1 = 0
✅
The Balance Check: 1 + 2 = 3 ✓ and 1 + 0 = 1 ✓

📝 Example 2: Rank & Nullity

Matrix: A = [1 2 3; 0 0 0; 0 0 0] (3×3)

Quick Analysis:

  • Pivots: Only at position (1,1)
  • Rank: 1
  • Nullity: 3 - 1 = 2
  • Column Space: The direction [1, 0, 0]^T
  • Null Space: The 2D plane perpendicular to [1, 2, 3]

📝 Example 3: Rank-2 Matrix

Matrix: A = [1 0; 0 1; 0 0] (3×2)

Rank = 2 Nullity = 2 - 2 = 0
  • Column Space: The xy-plane in 3D. Dimension = 2
  • Null Space: Only the zero vector. Dimension = 0
  • Row Space: Vectors [1,0] and [0,1]. Dimension = 2
  • Left Null Space: The z-direction. Dimension = 3 - 2 = 1
✅
Everything adds up perfectly: 2 + 0 = 2 ✓ and 2 + 1 = 3 ✓

💬 Frequently Asked Questions

What exactly is a dimension? Simple explanation? ▼

Simple Answer: Dimension is the number of independent choices needed to locate a point in a space.

Real examples:

  • A telephone pole: 1 dimension (up/down)
  • A chessboard: 2 dimensions (row, column)
  • Our world: 3 dimensions (length, width, height)

In linear algebra: Dimension = number of basis vectors in a space.

Why are they called "Four Fundamental Subspaces"? ▼

Because they completely describe what a matrix does. Together, they tell you:

  • What outputs are possible (column space)
  • What inputs vanish (null space)
  • The structure of rows (row space)
  • Hidden constraints (left null space)

They're the complete map of a matrix's behavior!

How do rank and dimension relate? ▼
  • Rank = number of pivot columns
  • Dimension of column space = rank
  • Dimension of row space = rank
  • Dimension of null space = n - rank

Key insight: Rank is the "master number" that unlocks all four subspace dimensions!

What's a basis? Why is it important? ▼

Definition: A minimal set of independent vectors that span a space.

Properties:

  • Independent (no overlaps)
  • Spanning (cover the entire space)
  • Minimal (can't remove any)
  • Number = dimension of space

Why it matters: A basis lets you represent any vector uniquely. It's your "coordinate system" for a space.

Can dimension be zero? What would that mean? ▼

Yes! Dimension is zero for the zero subspace {0}.

What this means: The only vector in the space is the zero vector.

When this happens:

  • If all columns are pivots, the null space is just {0}
  • If rank equals the number of rows, left null space is just {0}

It's rare but mathematically valid!

What does orthogonal (perpendicular) have to do with dimension? ▼

The beautiful truth: Row space and null space are perpendicular!

Why? If x is in null space, then Ax = 0. Every row of A is perpendicular to x.

What this means:

  • They're "opposite" directions
  • Their dimensions add up: rank + nullity = n
  • Together they fill the entire input space

Same principle: Column space ⊥ Left null space

How do I find the dimension of a subspace? ▼

Method:

  1. Reduce the matrix to row echelon form
  2. Count the pivot positions
  3. This count = dimension of column space

For null space:

  1. Count free variables (non-pivot columns)
  2. This count = dimension of null space

Quick check: (Rank) + (Nullity) = (Total columns) ✓

Why are pivot columns so important? ▼

Pivots are the "keys" to everything!

Once you count pivots (rank), you instantly know:

  • Dimension of column space = number of pivots
  • Dimension of row space = number of pivots
  • Dimension of null space = n - number of pivots
  • Which variables are free
  • The structure of all four subspaces

Why this works: Pivots identify the independent rows and columns—they show the "skeleton" of a matrix!
