Big Idea · Chapter 11

We study vector spaces through their building blocks (basis), and count how many independent directions (dimension) they have.

Every concept in this chapter — span, independence, basis, dimension, column space, null space — all builds toward this one goal: describe a space using the smallest possible set of independent directions.

The chain of ideas

Each concept depends on the one before it. Master them in this order:

1 Vector Space
2 Subspace
3 Linear Independence
4 Span
5 Basis
6 Dimension
7 Column / Null / Row Space

02 Vector Space vs Subspace

Start here. The whole chapter only makes sense once these two are clearly separated in your head.

Picture a vector space as the whole ocean. A subspace is a bay inside that ocean — it follows all the same rules, but lives inside the bigger space.

Vector Space (V)

  • A set of vectors satisfying all 8–10 axioms
  • Has closure, associativity, zero vector, etc.
  • It is the entire space
  • Exists independently

Subspace (W ⊆ V)

  • A subset of V that is itself a vector space
  • Must contain the zero vector $\mathbf{0}$
  • Closed under + and scalar multiplication
  • Depends on its parent space V
Every subspace IS a vector space. But not every vector space is a subspace (it has to live inside a larger space to be called a subspace).

03 Subspaces — The 3 Tests

How to tell whether a subset really is a subspace.

Subspace Test

A nonempty subset $W \subseteq V$ is a subspace if and only if all three hold:

1 $\mathbf{0} \in W$ (contains the zero vector)
2 $\mathbf{u}, \mathbf{v} \in W \Rightarrow \mathbf{u} + \mathbf{v} \in W$ (closed under +)
3 $\mathbf{u} \in W,\, c \in F \Rightarrow c\mathbf{u} \in W$ (closed under ·)
If any one of these fails, $W$ is NOT a subspace. The most common failure: forgetting that the zero vector must be inside.

The geometric pattern in $\mathbb{R}^n$

Subspaces of $\mathbb{R}^n$ are always "flat through the origin". They come in just a few flavors:

Zero subspace
$\{\mathbf{0}\}$ — just the origin
Line
span$\{\mathbf{v}\}$ through origin
Plane
span$\{\mathbf{v}, \mathbf{w}\}$ through origin
Whole space
$V$ itself

Common non-subspaces

  • $\{(x,y,z) : z = 1\}$ — a plane not through origin → fails ($\mathbf{0} \notin W$)
  • $\{(x,y,0) : x > 0\}$ — not closed under scalar multiplication (multiply by $-1$ → $x < 0$)
  • $\{(x,y,0) : x + y = 1\}$ — not closed under addition

04 Linear Dependence & Independence

Independence means "no redundancy" — every vector adds something genuinely new.

Linearly Dependent

There exist scalars $c_1, \ldots, c_k$ not all zero with:

$c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + \cdots + c_k\mathbf{v}_k = \mathbf{0}$

At least one vector is a combination of the others — redundancy.

Linearly Independent

The equation above has only the trivial solution:

$c_1 = c_2 = \cdots = c_k = 0$

No vector is built from the others — no redundancy.

How to check

3-step routine
  • Set up the equation $c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k = \mathbf{0}$
  • Write it as a linear system in $c_1, c_2, \ldots, c_k$
  • Solve. Only trivial solution → independent. Nontrivial solution exists → dependent.

Quick tests & facts

  • Two vectors: dependent iff $\mathbf{v}_1 = c\,\mathbf{v}_2$ for some scalar $c$
  • Three vectors in $\mathbb{R}^3$: dependent iff $\det[\mathbf{v}_1\ \mathbf{v}_2\ \mathbf{v}_3] = 0$
  • If $k > \dim(V)$: the set is automatically dependent
  • Zero vector kills independence: any set containing $\mathbf{0}$ is dependent
  • Single nonzero vector: always independent

05 Span

Everything you can build from a set of vectors using addition and scaling.

Definition

The span of $\mathbf{v}_1, \mathbf{v}_2, \ldots, \mathbf{v}_k$ in $\mathbb{R}^n$ is the set of all linear combinations of them:

$$\text{Span}\{\mathbf{v}_1, \ldots, \mathbf{v}_k\} = \{\, c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k \;\mid\; c_i \in \mathbb{R} \,\}$$

Geometric intuition

span{v₁}
A line through origin
span{v₁, v₂}
A plane (if independent)
span{v₁, v₂, v₃}
All of $\mathbb{R}^3$ (if independent)
Span is a "construction zone" — all the places you can reach by stretching and adding your given vectors.

Key properties of span

  • Span is always a subspace — it automatically passes the 3 tests
  • Span depends only on directions, not magnitudes
  • Adding a vector already in the span doesn't change the span
  • span$(\emptyset) = \{\mathbf{0}\}$ — empty set spans only the zero vector
  • span$\{\mathbf{v}_1, \ldots, \mathbf{v}_k\}$ is the smallest subspace of $V$ containing those vectors

How to check if a vector $\mathbf{b}$ is in span$(S)$

Procedure

Given $S = \{\mathbf{v}_1, \ldots, \mathbf{v}_k\}$ and a target $\mathbf{b}$:

  • Write $c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + \cdots + c_k\mathbf{v}_k = \mathbf{b}$
  • Solve for $c_1, c_2, \ldots, c_k$
  • Solution exists → $\mathbf{b} \in$ span$(S)$. No solution → $\mathbf{b} \notin$ span$(S)$.

06 Span & Basis Together

Span tells what you can build. Basis tells you the minimum tools to build it.

Span

  • "All linear combinations"
  • About reaching everywhere
  • Always a subspace
  • Can include redundant vectors

Basis

  • "Independent + spanning"
  • About no redundancy
  • The minimum spanning set
  • Every vector has a unique representation
The relationship: a basis is a spanning set that's also linearly independent — the smallest possible spanning set for the space.

07 Basis of a Subspace

The central concept of the chapter. Two simple rules — that's all a basis is.

Definition

Let $V$ be a subspace of $\mathbb{R}^n$. A basis of $V$ is a set of vectors $\{\mathbf{v}_1, \mathbf{v}_2, \ldots, \mathbf{v}_k\}$ in $V$ that satisfies:

01 Linearly Independent
02 Spans the space — $V = $ Span$\{\mathbf{v}_1, \ldots, \mathbf{v}_k\}$
Basis = Minimum + Independent + Spans
A basis is the smallest set of vectors that can produce every vector in the space. No redundancy, no missing pieces.

Three big facts

① Bases of $\mathbb{R}^n$ ↔ Invertible Matrices

Vectors $\mathbf{v}_1, \ldots, \mathbf{v}_n$ form a basis of $\mathbb{R}^n$ exactly when they are the columns of an $n \times n$ invertible matrix.

② $\mathbb{R}^n$ has Infinitely Many Bases

You can rotate, scale, or pick different independent vectors — endless valid bases exist.

③ Pivots Give You the Basis

  • Pivot columns of $A$ → basis for column space
  • Pivot rows of echelon form $R$ → basis for row space

Important properties

  • A basis of a subspace always consists of vectors in that subspace
  • All bases of a subspace have the same number of vectors (the dimension)
  • Removing any vector from a basis → it won't span anymore
  • Adding any vector to a basis → it won't be independent anymore
  • Basis is not unique, but the number of vectors is

08 Worked Examples — Checking Basis

The exam routine: test both properties one at a time. Three full examples below.

Example 1 ✓ Is a basis

Is $S = \left\{ \begin{bmatrix} 2 \\ 3 \end{bmatrix},\; \begin{bmatrix} 3 \\ 0 \end{bmatrix} \right\}$ a basis of $\mathbb{R}^2$?

Step 1 · Property 1: does $S$ span $\mathbb{R}^2$?

Solve $c_1 \begin{bmatrix} 2 \\ 3 \end{bmatrix} + c_2 \begin{bmatrix} 3 \\ 0 \end{bmatrix} = \begin{bmatrix} x_1 \\ x_2 \end{bmatrix}$, giving:

$2c_1 + 3c_2 = x_1$  (Eq. 1)     $3c_1 + 0c_2 = x_2$  (Eq. 2)

From Eq. 2: $c_1 = \dfrac{x_2}{3}$. Substitute into Eq. 1: $c_2 = \dfrac{x_1}{3} - \dfrac{2}{9}x_2$.

Solutions exist for any $x_1, x_2$ → Span$(S) = \mathbb{R}^2$ ✓

Step 2 · Property 2: linearly independent?

Set the combination equal to zero. From $3c_1 = 0$: $c_1 = 0$. Then $c_2 = 0$. Only the trivial solution ✓

Both properties hold → $S$ is a basis of $\mathbb{R}^2$.
Example 2 — The Standard Basis ✓ Is a basis

Is $S = \left\{ \begin{bmatrix} 1 \\ 0 \end{bmatrix},\; \begin{bmatrix} 0 \\ 1 \end{bmatrix} \right\}$ a basis of $\mathbb{R}^2$?

Span check

$c_1 \begin{bmatrix} 1 \\ 0 \end{bmatrix} + c_2 \begin{bmatrix} 0 \\ 1 \end{bmatrix} = \begin{bmatrix} x_1 \\ x_2 \end{bmatrix}$ → $c_1 = x_1,\ c_2 = x_2$. So Span $= \mathbb{R}^2$ ✓

Independence check

Setting equal to zero forces $c_1 = c_2 = 0$ ✓

$S$ is a basis — and it's the famous standard basis.
Example 3 — Too Many Vectors ✗ Not a basis

Is $S = \left\{ \begin{bmatrix} 2 \\ 3 \end{bmatrix},\; \begin{bmatrix} 3 \\ 0 \end{bmatrix},\; \begin{bmatrix} 1 \\ 0 \end{bmatrix} \right\}$ a basis of $\mathbb{R}^2$?

Independence check

Setting equal to zero: $3c_1 = 0 \Rightarrow c_1 = 0$. Plug back: $3c_2 + c_3 = 0 \Rightarrow c_2 = -\dfrac{c_3}{3}$.

So $c_2, c_3$ are not forced to be zero (e.g. $c_3 = 3, c_2 = -1$ works). Linearly dependent ✗

Property 2 fails → $S$ is not a basis of $\mathbb{R}^2$.
Pattern: $\mathbb{R}^2$ needs exactly 2 independent vectors. More than 2 → always dependent. Fewer than 2 → can't span. Same idea: $\mathbb{R}^n$ needs exactly $n$.

09 Standard Basis

The default coordinate system — your "rulers" along each axis.

Definition

In $\mathbb{R}^n$, the standard basis is the set of $n$ vectors $S = \{\mathbf{e}_1, \mathbf{e}_2, \ldots, \mathbf{e}_n\}$, where $\mathbf{e}_i$ has $1$ in the $i^{\text{th}}$ position and $0$ elsewhere.

For $\mathbb{R}^2$, written as $\hat{i}, \hat{j}$:

$\hat{i} = \begin{bmatrix} 1 \\ 0 \end{bmatrix}, \quad \hat{j} = \begin{bmatrix} 0 \\ 1 \end{bmatrix}$

Properties

  • The standard basis vectors are linearly independent ✓
  • They span $\mathbb{R}^n$ ✓
  • Therefore it's a basis of $\mathbb{R}^n$ — the simplest one
  • Any vector $\mathbf{x} = (x_1, x_2, \ldots, x_n)$ has unique representation $\mathbf{x} = x_1\mathbf{e}_1 + x_2\mathbf{e}_2 + \cdots + x_n\mathbf{e}_n$
  • The "transition matrix" from standard basis to itself is the identity matrix $I_n$
  • $\dim(\mathbb{R}^n) = n$ because the standard basis has $n$ vectors

10 Coordinates in a Basis

Same vector, different "address" depending on which basis you use.

Definition

Let $V$ be a vector space and $B = \{\mathbf{b}_1, \mathbf{b}_2, \ldots, \mathbf{b}_n\}$ a basis of $V$. Every $\mathbf{v} \in V$ can be written uniquely as:

$\mathbf{v} = c_1\mathbf{b}_1 + c_2\mathbf{b}_2 + \cdots + c_n\mathbf{b}_n$

The coordinate vector of $\mathbf{v}$ relative to $B$ is the column of scalars:

$[\mathbf{v}]_B = \begin{bmatrix} c_1 \\ c_2 \\ \vdots \\ c_n \end{bmatrix}$

How to find $[\mathbf{v}]_B$

4 steps
  • Let $B = \{\mathbf{b}_1, \ldots, \mathbf{b}_n\}$ be a basis of $V$
  • Write $\mathbf{v} = c_1\mathbf{b}_1 + \cdots + c_n\mathbf{b}_n$
  • Solve for the scalars $c_1, \ldots, c_n$
  • Stack them into the coordinate vector $[\mathbf{v}]_B$
Example in $\mathbb{R}^2$ Worked Solution

Let $B = \{\mathbf{b}_1, \mathbf{b}_2\}$ where $\mathbf{b}_1 = \begin{bmatrix} 1 \\ 1 \end{bmatrix},\, \mathbf{b}_2 = \begin{bmatrix} 2 \\ -1 \end{bmatrix}$ and $\mathbf{v} = \begin{bmatrix} 7 \\ 1 \end{bmatrix}$. Find $[\mathbf{v}]_B$.

Set up

$c_1\begin{bmatrix}1\\1\end{bmatrix} + c_2\begin{bmatrix}2\\-1\end{bmatrix} = \begin{bmatrix}c_1 + 2c_2\\c_1 - c_2\end{bmatrix} = \begin{bmatrix}7\\1\end{bmatrix}$

Solve the system

$c_1 + 2c_2 = 7$  and  $c_1 - c_2 = 1$ → subtract: $3c_2 = 6 \Rightarrow c_2 = 2$, then $c_1 = 3$.

$[\mathbf{v}]_B = \begin{bmatrix} 3 \\ 2 \end{bmatrix}$

Key facts

  • Coordinates depend on the basis chosen — same vector, different coordinates in different bases
  • If $B$ is the standard basis, $[\mathbf{v}]_B$ is just the usual column form of $\mathbf{v}$
  • The coordinate map $\mathbf{v} \mapsto [\mathbf{v}]_B$ is a 1-to-1 correspondence between $V$ and $\mathbb{R}^n$
  • $[a\mathbf{v} + b\mathbf{w}]_B = a[\mathbf{v}]_B + b[\mathbf{w}]_B$ (linear)
  • $[\mathbf{0}]_B = \mathbf{0}$

11 Change of Basis

Translate the same vector's address from one basis to another.

Definition

Let $B$ and $C$ be two bases of $V$. The change of basis matrix $P_{C \leftarrow B}$ converts coordinates from $B$ to $C$:

$[\mathbf{v}]_C = P_{C \leftarrow B}\,[\mathbf{v}]_B$

Building $P_{C \leftarrow B}$

Express each $\mathbf{b}_i$ (vector in basis $B$) as a combination of $C$'s vectors. The coefficients become the columns of $P_{C \leftarrow B}$.

Important inverse relationship:   $P_{B \leftarrow C} = (P_{C \leftarrow B})^{-1}$

Key properties

  • $P_{B \leftarrow B} = I$ (identity)
  • Change of basis matrices are always invertible
  • $P_{C \leftarrow B} \cdot P_{B \leftarrow C} = I$
  • Chaining: $P_{C \leftarrow B} \cdot P_{B \leftarrow D} = P_{C \leftarrow D}$ (go via $B$)
  • Columns of $P_{C \leftarrow B}$ = coordinates of $C$-vectors in terms of $B$

12 Dimension

A single number that captures the "size" of a space — the count of independent directions.

Definition

The dimension of a vector space $V$, denoted $\dim(V)$, is the number of vectors in any basis of $V$. Every basis of the same space has the same count (Replacement Theorem / Exchange Lemma).

Case 1 · The Plane $\mathbb{R}^2$

A flat sheet of paper.

  • 2 vectors (one →, one ↑) → independent & span ✓
  • 1 vector → only a line, can't span ✗
  • 3 vectors → one is redundant ✗

$\dim \mathbb{R}^2 = 2$

Case 2 · The Space $\mathbb{R}^3$

3D space (the real world).

  • 3 vectors (←→, ↕, ⤴) → ✓
  • 2 vectors → only a plane ✗
  • 4 vectors → one redundant ✗

$\dim \mathbb{R}^3 = 3$

Quick formulas

$\dim(\mathbb{R}^n)$
$= n$
$\dim(\{\mathbf{0}\})$
$= 0$
$\dim(M_{m \times n})$
$= mn$
$\dim(P_n)$
$= n+1$ (polynomials)

Important theorems

  • If $V$ is finite-dimensional and $S$ is a spanning set → $|S| \geq \dim(V)$
  • If $V$ is finite-dimensional and $S$ is linearly independent → $|S| \leq \dim(V)$
  • If $S$ is both spanning and independent → $S$ is a basis and $|S| = \dim(V)$
  • If $W \subseteq V$ is a subspace → $\dim(W) \leq \dim(V)$
  • If $V$ has a basis with $k$ vectors → any set of more than $k$ vectors in $V$ is dependent
How to find dimension (shortcut): if you can find $k$ linearly independent vectors that span the space, then $\dim = k$.

13 Column Space C(A)

"All possible outputs" when you feed inputs into matrix $A$.

Definition

The column space of an $m \times n$ matrix $A$, denoted $C(A)$ or Col$(A)$, is the span of the columns of $A$. It's the set of all possible outputs $\mathbf{b}$ such that $A\mathbf{x} = \mathbf{b}$ has a solution.

$C(A) \subseteq \mathbb{R}^m$ (a subspace of the output space)

Think of $A$ as a machine. The column space is everything the machine can produce.

Key facts

  • $C(A)$ is a subspace of $\mathbb{R}^m$ (the output space)
  • $\dim(C(A)) = \text{rank}(A)$
  • The pivot columns of $A$ form a basis of $C(A)$ (always pick from the original $A$, not from $R$)
  • $A\mathbf{x} = \mathbf{b}$ is solvable $\iff \mathbf{b} \in C(A)$
  • If $A$ has no pivot ($A = 0$) → $C(A) = \{\mathbf{0}\}$, basis is empty

14 Null Space N(A)

The "hidden" inputs that the matrix $A$ collapses to zero.

Definition

The null space of $A$, denoted $N(A)$ or Nul$(A)$, is the set of all $\mathbf{x} \in \mathbb{R}^n$ such that $A\mathbf{x} = \mathbf{0}$.

$N(A) = \{ \mathbf{x} \in \mathbb{R}^n \mid A\mathbf{x} = \mathbf{0} \}$

All vectors in $N(A)$ collapse to the zero vector after applying $A$. They represent "lost information".

Key facts

  • $N(A)$ is a subspace of $\mathbb{R}^n$ (the input space)
  • $\dim(N(A)) = \text{nullity}(A) = n - \text{rank}(A)$
  • $N(A)$ always contains the zero vector
  • If $N(A) = \{\mathbf{0}\}$ → no "hidden" directions, $A$ is one-to-one
  • If $A$ is invertible → $N(A) = \{\mathbf{0}\}$ only

How to find a basis for $N(A)$

Procedure
  • Row reduce $A$ to row echelon form $R$
  • Solve $R\mathbf{x} = \mathbf{0}$ (same solutions as $A\mathbf{x} = \mathbf{0}$)
  • Express solutions using free variables
  • For each free variable, set it $= 1$ (others $= 0$) → that solution vector is a basis vector

16 Basis for Column Space — Pivot Columns

The clean method: row reduce, find pivots, take those original columns.

Method

4 steps
  • Row reduce $A$ to row echelon form $R$
  • Identify the pivot columns in $R$
  • Take the corresponding columns from the original matrix $A$ (not from $R$!)
  • Those columns form a basis for $C(A)$
Always pick columns from the original matrix $A$, not from the row echelon form $R$. Row operations change the column space — but they don't change which columns are pivots.

Why this works

  • Pivot columns are linearly independent
  • Every other column is a linear combination of the pivot columns
  • So the pivot columns span $C(A)$ and are independent → they form a basis

Rank connection

Number of pivots
$= \text{rank}(A)$
Number of pivots
$= \dim(C(A))$
Number of pivots
$=$ max # of independent columns of $A$

17 Row Space & Rank–Nullity Theorem

The third fundamental subspace, plus the equation that ties everything together.

Row Space

Definition

The row space of an $m \times n$ matrix $A$, denoted Row$(A)$, is the subspace of $\mathbb{R}^n$ spanned by the rows of $A$.

How to find a basis for Row$(A)$

  • Row reduce $A$ to row echelon form $R$
  • The nonzero rows of $R$ form a basis for Row$(A)$
  • Reason: row operations don't change the row space, and the nonzero rows of $R$ are linearly independent

The big result: Rank–Nullity

For any $m \times n$ matrix $A$:

$\dim(C(A)) + \dim(N(A)) = n$

rank$(A) +$ nullity$(A) =$ number of columns

Summary of fundamental subspaces

SubspaceLives inBasis fromDimension
Row$(A)$$\mathbb{R}^n$Nonzero rows of $R$rank$(A)$
Col$(A)$$\mathbb{R}^m$Pivot columns of $A$rank$(A)$
Nul$(A)$$\mathbb{R}^n$Solution vectors from free variables$n - $ rank$(A)$
Row space and column space have the same dimension (both equal rank). That's not a coincidence — it's a deep theorem about matrices.

18 Matrix Operations — Quick Reference

The algebra behind everything else in this chapter.

The three basic operations

Addition
$A + B = [a_{ij} + b_{ij}]$ — same size only
Scalar multiplication
$cA = [c\,a_{ij}]$
Matrix multiplication
$AB = C$ where $c_{ij} = \sum_k a_{ik} b_{kj}$ — needs $A_{m\times n} B_{n\times p}$

Key rules

  • $A + B = B + A$ (commutative)
  • $(A + B) + C = A + (B + C)$ (associative)
  • $A(BC) = (AB)C$ (associative)
  • $A(B + C) = AB + AC$ (distributive)
  • $AB \neq BA$ in general — matrix multiplication is NOT commutative
  • $AI_n = A = I_m A$ — identity matrix is neutral
  • $A\mathbf{0} = \mathbf{0}A = \mathbf{0}$
Always check dimensions before multiplying: for $AB$ to exist, columns of $A$ = rows of $B$. If $A$ is $m \times n$ and $B$ is $n \times p$, then $AB$ is $m \times p$.

19 Determinants

One number per square matrix that tells you about invertibility, volume scaling, and independence.

Definition

For an $n \times n$ matrix $A$, the determinant $\det(A)$ (or $|A|$) is a scalar that satisfies:

$A$ is invertible $\iff \det(A) \neq 0$

Formulas

2×2 matrix

For $A = \begin{bmatrix} a & b \\ c & d \end{bmatrix}$:

$\det(A) = ad - bc$

3×3 matrix (cofactor expansion)

For $\begin{bmatrix} a & b & c \\ d & e & f \\ g & h & i \end{bmatrix}$:

$\det = a(ei - fh) - b(di - fg) + c(dh - eg)$

Geometric meaning

  • $|\det(A)|$ = factor by which $A$ scales $n$-dimensional volume
  • $\det(A) > 0$ → orientation preserved
  • $\det(A) < 0$ → orientation reversed
  • $\det(A) = 0$ → space collapses to a lower dimension (matrix is singular)

Row operations & determinants

OperationEffect on det
Swap two rowsMultiply by $-1$
Multiply a row by $k$Multiply by $k$
Add multiple of one row to anotherNo change

Useful properties

  • $\det(I_n) = 1$
  • $\det(A^T) = \det(A)$
  • $\det(AB) = \det(A)\det(B)$
  • $\det(kA) = k^n \det(A)$ for $n \times n$ matrix
  • If $A$ is triangular → $\det(A)$ = product of diagonal entries
  • If two rows (or columns) are equal → $\det(A) = 0$
  • If a row (or column) is all zeros → $\det(A) = 0$

20 Matrix Spaces

Matrices themselves can form vector spaces. Same rules — just bigger objects.

The space of all 2×2 matrices

Called $M$ (or $M_{2 \times 2}$). A natural basis:

$A_1 = \begin{bmatrix} 1 & 0 \\ 0 & 0 \end{bmatrix},\; A_2 = \begin{bmatrix} 0 & 1 \\ 0 & 0 \end{bmatrix},\; A_3 = \begin{bmatrix} 0 & 0 \\ 1 & 0 \end{bmatrix},\; A_4 = \begin{bmatrix} 0 & 0 \\ 0 & 1 \end{bmatrix}$

Any 2×2 matrix is $c_1 A_1 + c_2 A_2 + c_3 A_3 + c_4 A_4$. So they span. Independence is obvious: the only way to get the zero matrix is all $c_i = 0$.

$\dim(M_{2\times 2}) = 4$

Dimension table

SpaceWhat it containsDimension
$M_{m \times n}$All $m \times n$ matrices$mn$
$M_n$All $n \times n$ matrices$n^2$
$D_n$All $n \times n$ diagonal matrices$n$
$U_n$All $n \times n$ upper triangular$\dfrac{n(n+1)}{2}$
$L_n$All $n \times n$ lower triangular$\dfrac{n(n+1)}{2}$
$S_n$All $n \times n$ symmetric$\dfrac{n(n+1)}{2}$
$K_n$All $n \times n$ skew-symmetric$\dfrac{n(n-1)}{2}$

Where does $\dfrac{n(n+1)}{2}$ come from?

For an upper triangular matrix, only entries on and above the diagonal can vary. Count row by row:

  • Row 1: $n$ entries
  • Row 2: $n-1$ entries
  • Row 3: $n-2$ entries
  • …
  • Row $n$: $1$ entry

Total: $n + (n-1) + \cdots + 1 = \dfrac{n(n+1)}{2}$ — the famous Gauss sum.

Subspace relationships

  • $D_n \subset U_n$ and $D_n \subset L_n$ (diagonal is inside upper & lower triangular)
  • $U_n \cap L_n = D_n$ (only diagonal matrices are both)
  • $U_n + L_n = M_n$ (every matrix = upper + lower − diagonal)
  • $S_n \subset M_n$, $K_n \subset M_n$, $D_n \subset S_n$

$\mathbf{0} \in W$, closed under $+$ and $\cdot$
Span
All linear combinations $c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k$
Lin. independence
Only trivial solution to $c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k = \mathbf{0}$
Basis
Independent + spans the space
Dimension
Number of vectors in any basis
$\dim(\mathbb{R}^n)$
$= n$
Standard basis
$\{\mathbf{e}_1, \ldots, \mathbf{e}_n\}$ — 1 in $i^{th}$ slot
$[\mathbf{v}]_B$
Coordinates of $\mathbf{v}$ in basis $B$
Change of basis
$[\mathbf{v}]_C = P_{C \leftarrow B}\,[\mathbf{v}]_B$
$P_{B \leftarrow C}$
$= (P_{C \leftarrow B})^{-1}$
Column space
Span of columns of $A$; basis = pivot cols of $A$
Null space
$\{\mathbf{x} : A\mathbf{x} = \mathbf{0}\}$; basis from free variables
Row space
Span of rows; basis = nonzero rows of $R$
Rank–Nullity
$\text{rank}(A) + \text{nullity}(A) = n$
$\dim(C(A))$
$= \dim(\text{Row}(A)) = \text{rank}(A)$
$\det(A) \neq 0$
$\iff A$ is invertible
$\det(AB)$
$= \det(A)\det(B)$
$\dim(M_{m\times n})$
$= mn$
Upper triangular dim
$\dfrac{n(n+1)}{2}$
Exam routine
Row reduce → find pivots & free vars → write basis → compute dim
