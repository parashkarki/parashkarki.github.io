Key Formulas:
• \(L_1\) Norm (Taxicab Distance): \(||x||_1 = |x_1| + |x_2| + \dots + |x_n|\)
• \(L_2\) Norm (Euclidean Distance): \(||x||_2 = \sqrt{x_1^2 + x_2^2 + \dots + x_n^2}\)
• \(L_\infty\) Norm (Maximum Norm): \(||x||_\infty = \max(|x_1|, |x_2|, \dots, |x_n|)\)
• Normalization (using \(L_2\)): \(u = \frac{v}{||v||_2}\)
Part I: Vector Construction and Notation
Question 1: Movement to Column Vector

In Linear Algebra, we define a vector as a specific instruction for displacement. Given the instruction: "Walk 4 units Left and 7 units Up," translate this movement into a vertical column vector in the format \(\begin{bmatrix} x \\ y \end{bmatrix}\).

Answer:
By convention, "Left" corresponds to a negative x-displacement and "Up" corresponds to a positive y-displacement.
\[ \vec{v} = \begin{bmatrix} -4 \\ 7 \end{bmatrix} \]
Question 2: Standard Basis Conversion

A vector is written in standard basis notation as \(3i - 2j\). Provide the equivalent vertical column vector.

Answer:
Using standard basis vectors \(i = \begin{bmatrix} 1 \\ 0 \end{bmatrix}\) and \(j = \begin{bmatrix} 0 \\ 1 \end{bmatrix}\):
\[ 3i - 2j = 3\begin{bmatrix} 1 \\ 0 \end{bmatrix} - 2\begin{bmatrix} 0 \\ 1 \end{bmatrix} = \begin{bmatrix} 3 \\ -2 \end{bmatrix} \]
Part II: Calculating Vector Norms
Question 3: The Taxicab Distance (\(L_1\) Norm)

Calculate the \(L_1\) norm for the vector \(v = \begin{bmatrix} -5 \\ 12 \end{bmatrix}\).

Answer:
The \(L_1\) norm is the sum of the absolute values of the components:
\[ ||v||_1 = |-5| + |12| = 5 + 12 = 17 \]
Question 4: The Euclidean Norm (\(L_2\) Norm)

Calculate the \(L_2\) norm for the same vector \(v = \begin{bmatrix} -5 \\ 12 \end{bmatrix}\).

Answer:
The \(L_2\) norm is the square root of the sum of squares:
\[ ||v||_2 = \sqrt{(-5)^2 + 12^2} = \sqrt{25 + 144} = \sqrt{169} = 13 \]
Question 5: The Maximum Norm (\(L_\infty\) Norm)

Calculate the \(L_\infty\) norm for the vector \(W = \begin{bmatrix} 8 \\ -10 \end{bmatrix}\).

Answer:
The \(L_\infty\) norm is the maximum absolute value among the components:
\[ ||W||_\infty = \max(|8|, |-10|) = \max(8, 10) = 10 \]
Question 6: Comparative Norm Analysis

For the vector \(a = \begin{bmatrix} 3 \\ -4 \end{bmatrix}\), calculate \(L_1, L_2,\) and \(L_\infty\) norms. Rank them in descending order to verify the property \(L_1 \ge L_2 \ge L_\infty\).

Answer:
1. \(||a||_1 = |3| + |-4| = 7\)
2. \(||a||_2 = \sqrt{3^2 + (-4)^2} = \sqrt{25} = 5\)
3. \(||a||_\infty = \max(|3|, |-4|) = 4\)
Ranking: \(7 \ge 5 \ge 4\). This confirms the property \(L_1 \ge L_2 \ge L_\infty\).
Part III: Unit Vectors and Normalization
Question 7: Unit Vector Definition

If a vector \(u\) satisfies the property \(u \cdot u = 1\), what is its \(L_2\) magnitude? Furthermore, what specific term do we use to describe a vector that possesses this magnitude?

Answer:
Since \(||u||_2 = \sqrt{u \cdot u}\), the magnitude is \(\sqrt{1} = 1\). The specific term for such a vector is a Unit Vector.
Question 8: Performing Normalization

Given the vector \(X = \begin{bmatrix} 6 \\ 8 \end{bmatrix}\), find the unit vector \(u\) that points in the same direction.

Answer:
1. Step 1: Calculate \(||X||_2 = \sqrt{6^2 + 8^2} = \sqrt{100} = 10\).
2. Step 2: Divide components by magnitude: \(u = \frac{1}{10}\begin{bmatrix} 6 \\ 8 \end{bmatrix} = \begin{bmatrix} 0.60 \\ 0.80 \end{bmatrix}\).
3. Conceptual Check: The direction did not change because normalization involves scalar multiplication by a positive constant (\(1/||X||_2\)), which preserves orientation.
Question 9: Normalization with Negative Components

Normalize the vector \(b = \begin{bmatrix} -3 \\ 0 \end{bmatrix}\). Provide your answer in vertical column vector format.

Answer:
1. Calculate magnitude: \(||b||_2 = \sqrt{(-3)^2 + 0^2} = 3\).
2. Normalize: \(u = \frac{1}{3}\begin{bmatrix} -3 \\ 0 \end{bmatrix} = \begin{bmatrix} -1.00 \\ 0.00 \end{bmatrix}\).
Question 10: Verification of Normalization

Take your resulting unit vector \(u\) from Question 8. Prove that it is a unit vector by calculating its \(L_2\) norm. Explicitly show the SRSS (Square Root of the Sum of Squares) steps.

Answer:
Vector \(u = \begin{bmatrix} 0.60 \\ 0.80 \end{bmatrix}\).
Verification using SRSS:
\[ ||u||_2 = \sqrt{(0.60)^2 + (0.80)^2} \] \[ ||u||_2 = \sqrt{0.36 + 0.64} \] \[ ||u||_2 = \sqrt{1.00} = 1.00 \] The result equals 1, confirming \(u\) is a unit vector.
